Server-side (/l2c-34-oof/software/server/)
- To host server
    - cd into server directory
    - run the command 'python server.py'

Clint-side (/l2c-34-oof/software/client/)
- To open client connection (*must use Linux*):
    - cd into client directory
    - compile c code with 'gcc client.c -o client'
    - run the command './client'

Game (/l2c-34-oof/software/game/)
- To run game
    - cd into server directory
    - run the command 'python main.py'
