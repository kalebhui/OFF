General:
- r:        reset level

player 1 keys:
- w:        jump
- a:        move left
- d:        move right

player 2 keys:
- i:        move up
- k:        move down
- j:        move left
- l:        move right
- 1:        select normal block
- 2:        select ice block
- 3:        select trampoline block
- space:    place block
